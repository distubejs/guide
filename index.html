# DisTube

# Features

  • Support YouTube, SoundCloud, Facebook, and 700+ more sites (opens new window)
  • Audio filters (bassboost, nightcore, vaporwave,...)
  • Build on discord.js v12
  • Easily to use and customize
  • Multiple servers compatible

# Installation

npm install distube

# Requirement

  • node (opens new window) >= 14.0.0
  • discord.js (opens new window) v12
  • FFmpeg (opens new window) - npm install ffmpeg-static
  • @discordjs/opus (opens new window) - npm install @discordjs/opus
  • python (opens new window) (For youtube-dl (opens new window) to support 700+ more sites (opens new window).)
